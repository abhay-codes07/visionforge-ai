# Visionary Agent Protocol

Production-grade real-time Vision AI Agent platform.

## Monorepo

- `frontend/`: Next.js 14 luxury UI and realtime client.
- `backend/`: FastAPI REST/WebSocket vision services.
- `docker/`: Compose and Dockerfiles for local/prod-like runtime.
- `docs/`: Architecture, API, AI integration, and deployment runbooks.

## Quick Start

- Frontend dev: `npm run dev:frontend`
- Backend dev: `npm run dev:backend`
- Backend tests: `npm run test:backend`

## Container Ops

- Start stack: `npm run up`
- Stop stack: `npm run down`
- Follow logs: `npm run logs`
